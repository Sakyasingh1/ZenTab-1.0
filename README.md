> **A controlled digital workspace designed for deep focus, structured learning, intentional browsing, and uncompromising user privacy.**

ZenTab is not just a browser; it's a productivity powerhouse. Engineered for students, researchers, and deep workers, it combines an intelligent browsing environment with multi-provider AI assistance, an encrypted vault, and a strict focus system.

---

## ✨ Key Features

### 🧠 ZenAI Bridge
Multi-provider AI assistant integrated directly into your workflow. Support for:
- **Anthropic Claude 3.5**
- **OpenAI GPT-4o**
- **Google Gemini 1.5 Pro**
- **Ollama (Local & Private)**
- **DeepSeek & Qwen 2.5**

### 🧘 Focus Engine
Built-in Pomodoro timer with immersive ambient soundscapes (Rain, Lo-fi, Forest) and strict mode to keep you on task.

### 🛡️ Secure Shield
8-layer security model including:
- **Phishing Shield (ML-powered)**
- **Tracker & Ad Blocker**
- **DNS over HTTPS (DoH)**
- **AES-256 Encrypted Vault**

### 📊 Real-time Analytics
Track your productivity with beautiful charts and a dynamic "Focus Score" that reflects your intentional work habits.

## 🚀 Quick Start

### 1. Clone the Repository
```bash
git clone https://github.com/Sakyasingh1/ZenTab-1.0.git
cd ZenTab-1.0
```

### 2. Run Locally
Simply open `index.html` in any modern browser, or use a live server:
```bash
npx live-server
```

## 🏗️ Technology Stack

- **Core**: HTML5, Vanilla JavaScript (ES6+), CSS3
- **Design**: Custom CSS Variables, Glassmorphism, CSS Animations
- **Storage**: Browser LocalStorage (Prototype) / SQLCipher (Native)
- **AI**: Multi-Provider REST API Integration
